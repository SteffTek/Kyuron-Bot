Kyuron - Help Page

Kyuron is fully modular. But this also means there are a TON of commands to learn and master!

Modules

About:

Modules are the heart of Kyuron. You can enable or disable any behavior this bot does.

Commands:

● /module module nameenabled Enables or disables the specified module for this guild.

List of Modules:

Pro Tip: Click on one to jump directly to it!

● Audit Logging Logs everything to a specified channel. Like message deletion, channel creation, etc.
● Auto Responder Responses with a predefined message if a user asks something.
● Auto Mod Automatically hands out warns or bans users based on a guild defined preset.
● Auto Voice Channel Creates Voice Channel on the fly and keeps them open as long as a user is in them.
● Announcements Notifies you of new uploads or live streams of your favorite content creators.
● Economy (WIP) ???
● F.U.N. Share Memes, give out cookies, or ship someone! It's F.U.N.

● Leveling (WIP) Level up your chatting skills and earn XP by typing.
● Moderation Basic crowd control functions for your discord!
● Tickets Ticket system. If someone on your discord has a problem, let them ask it in a channel dedicated for him/her!
● Polls Create basic polls or surveys with this module.
● Reaction Roles Automatically hand out roles based on a reaction the users gives!

● Warn An addon for the Audit Logger. Warns if a user that joins has an account newer than 3 days or is banned on a global ban database.

Moderation

About:

The moderation module allows you and your moderators to take action on your guild.

Commands:

● /kick user | optionalreason Kick a user for the specified reason.
● /ban user | optionalreason Ban a user for the specified reason.
● /warn userreason Warns a user for the specified reason.
● /unban user Unban a user previously banned.
● /tempban userdurationdurationtype | optionalreason Temporarily bans a user for the specified duration.
● /mute user | optionalreason Mutes a user for the specified reason.
● /unmute user Unmute a user previously muted.
● /tempmute userdurationdurationtype | optionalreason Temporarily mutes a user for the specified duration.
● /clear amount Clears the last X amount of messages in the current channel.
● /modlog user | optionalpage Shows the mod actions for the selected user.
● /userinfo user Shows useful information about a user.
● /blacklist setterm Blocks a word from usage on your discord guild.
● /blacklist remove term Permits a blocked word to be used again.
● /blacklist get Lists all blocked words.

Auto Mod

About:

Auto Mod helps you moderate your server. You can set up automatic actions and trigger events if a user got to many infractions.


Triggers help you, for example, automatically mute someone after he got 5 warns in the last 30 minutes.

Example:/automod trigger set Mute 5 30

Commands:

● /automod rules enabled ruleenabled Enables or disables a rule.
● /automod rules action ruleaction What should be done in case a rule is broken.
● /automod rules whitelist rulechannel Whitelist channels where the rule is allowed to be broken.
● /automod trigger get List all triggers you have active. You can have up to 5 triggers.
● /automod trigger remove trigger Remove a trigger by its ID. Obtain the ID via trigger get.
● /automod trigger set whatwarnstimespan Creates a trigger. Duration and duration type are needed for temp mutes/bans.

Auto Responder

About:

Getting annoyed by frequently asked questions? Let the auto responder automatically answer for you!

Notice: You can have up to 25 responses per guild.

Commands:

● /responder get Get all responses active in the current channel.
● /responder remove name Removes the responses specified by the name.
● /responder set namequestionresponse Create a response by the specified name.
The question will be stripped of any punctuation marks.

Announcements

About:

Announcements provide an opportunity to create notifications for social media platforms. Live notifications are available for Twitch, YouTube and Twitter.

Notice: Kyuron checks every 5 minutes for new content to keep server load at a minimum. If your notification doesn't get displayed immediately, please wait a bit longer.

Commands:

● /announcement set platformurlmessage Creates a new announcement in the current channel.
● /announcement remove url Removes the announcement specified by the url.

Available placeholders for the messages are %user% and %link%.

Reaction Roles

About:

With reaction roles you can give your users roles based on their reactions!

Commands:

● /reactionrole create name Creates a new reaction role in this channel with the given name.
● /reactionrole remove name Removes the reaction role specified by the name.
● /reactionrole addrole nameroleemote Adds the selected role to the specified reaction with the given emote.
● /reactionrole removerole namerole Removes the selected role from the specified reaction.

Polls

About:

The poll module helps with interactive polls on your discord guild.

Notice: If the bot crashes, ongoing polls will be lost :/

Commands:

● /poll vote text | optionaltimer (in minutes) Creates a Yes/No poll in the current channel.
The winner will be announced after the timer expires.
● /poll survey text option 1option 2 | optionaloption 3-10timer Creates a survey with the provided options.
The winner will be announced after the timer expires.

Giveaway

About:

With the giveaway module you can create simple giveaways. User can react to the message to join.

Notice: If the bot crashes, ongoing giveaways will be lost :/

Commands:

● /giveaway pricetimer Creates a giveaway for the specified price and time in minutes.

Audit Logger

About:

The audit logger logs almost every event happening on your server, and that can be quite a lot!

Notice: Can be customized with /setchannel

Warn

About:

The warn module is an addition to the audit logger. If a user joins it checks if the account is newer than 3 days and sends an audit log. Additionally: On of the creators of Kyuron has another bot which protects guilds from bot raids. It has an internal database of potentially harmful users, which is also checked against.

Leave Notice

About:

Notify users if a users leaves the guild.

Notice: Can be customized with /setchannel and /setmessage

Auto Voice Channel

About:

Automatically create and delete voice channels based on need. A user just needs to join your dedicated voice channel and he will automatically sent to a channel just for him. If everyone leaves the dedicated channel, it will be automatically deleted. Never run out of voice channels again!

Notice: If the bot crashes, channels will not be deleted!

Further notice: Can be customized with /setchannel

Ticket System

About:

The ticket system helps your users with private issues only a moderator can solve. It creates a channel for every ticket a user requests. On finish, you can easily delete the channel.

Notice: The ticket system is automatically created on module enable and managed by Kyuron.

Further notice: Can be customized with /setmessage

Misc

Commands that not belong to a specific module.

Commands:

● /setchannel type | optionalchannel There are different roles for different channels.
You can set specific channel tasks with this command.
● /setmessage typemessage Like /setchannel, different modules use predefined messages.
Those are set with this command.
● /role get get/set/remove type and/or role Various commands around specific roles like mod roles and the mute role.

Available placeholders for the messages are %user%.
